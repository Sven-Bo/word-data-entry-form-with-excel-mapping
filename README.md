#  Data Entry Form to generate MS Word Document

Data entry form to generate Word document. GUI will retrieve key-value pairs from an Excel file based on number input.


## Demo
![Demo](/demo.jpg "Demo")

## Run the GUI
```
# vanilla terminal
python data_entry_word.py
```

## Requirements
```
openpyxl==3.0.10
pandas==1.3.5
docxtpl==0.11.4
PySimpleGUI==4.60.1
```
